данное приложение формирует отчет на основе переданных файлов с данными

чтобы сформировать отчет, нужно передать два метода, через консоль
python main.py --files economic1.csv economic2.csv --report gdp_avg

--files <укажите файлы на основе которых вы хотите составить отчет>
--report <укажите какой именно отчет вы хотите составить(базово стоит gdp_avg)>

в архитектуру заложена возможность добовления новых отчетов
достаточно создать новый класс, на основе BaseReport
в него уже заложенны эти функции:
-- scanning_data считывает один файл и передает все данные ввиде словоря
-- report_to_file может записать сформированный отчет в отдельный файл(этого функционала не было в задании, моя инициатива)(по умолчанию эта функция отключена)
